## ✨ **Overview**
This project demonstrates how to implement **Redux Toolkit** for state management in a **React** app.  
Users can:
- 🛒 Add products to cart  
- 🔄 Update quantities dynamically  
- ❌ Remove items  
- 🧹 Clear the entire cart  
- 💸 See total price update instantly
